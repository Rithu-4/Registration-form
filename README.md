##** Registration Form with Validation**
This project is a simple and responsive **Registration Form** created using **HTML, CSS, and JavaScript**.  
The form includes **client-side validation** to check the user inputs before submitting the form.

** Features**
- User-friendly registration form
- Name field validation
- Email format validation
- Password validation
- Error messages for invalid inputs
- Clean and responsive design

## **Technologies Used**
- HTML
- CSS
- JavaScript

##** Project Structure**
registration-form/

├── index.html
├── style.css
├── script.js


## **How It Works**
1. The user enters their name, email, and password.
2. JavaScript checks if the inputs are valid.
3. If any field is incorrect, an error message is shown.
4. If all inputs are valid, the form is successfully submitted.

## **Purpose of the Project**
This project was created to practice **form validation using JavaScript** and to understand how front-end forms work in web development.
